# 开发路线图

## 第 1 阶段：可运行骨架

- Vue 3 PWA 外壳
- 记录、时间线、日历、心情、设置页面
- Fastify API 服务
- 邮箱验证码接口骨架
- 同步接口骨架
- 文件上传接口骨架
- AI 服务接口骨架

## 第 2 阶段：本地优先体验

- IndexedDB 持久化日记和草稿
- 离线创建草稿
- 同步队列
- 图片缓存
- 导出日记卡片图片
- JSON 备份导出与导入

## 第 3 阶段：账号与云同步

- 邮箱验证码登录骨架
- 手动上传、拉取、双向同步
- PostgreSQL `sync_diaries` 持久表
- 数据库不可用时内存兜底
- 后续接真实邮件验证码和完整用户表

## 第 4 阶段：AI 接入

- 主体点选/框选参数上传
- 抠图服务适配
- 风格化服务适配
- 日记生成服务适配
- 处理队列与失败重试

## 第 5 阶段：贴纸编辑器完善

- 多贴纸画布
- 贴纸拖动、缩放、旋转
- 文字贴纸
- 装饰贴纸
- 画笔涂鸦
- 自动排版模板
- 详情页预览与 PNG 导出

## 第 6 阶段：部署

- Docker Compose 生产配置
- HTTPS 与反向代理
- 存储目录备份
- 数据库备份
- 日志与健康检查
